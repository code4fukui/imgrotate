# imgrotate

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A minimalist, client-side web app for rotating images.

## Demo

**~~https://code4fukui.github.io/img~~ *(unavailable)*rotate/**

The interface features a large image area and a single slider for control.

## Features

-   **Drag & Drop:** Instantly load an image by dragging and dropping it onto the page.
-   **Slider Control:** Precisely adjust the image's rotation using a simple slider.
-   **Wide Rotation Range:** Rotate images from -100 to +100 degrees.
-   **Client-Side Processing:** All processing is done in your browser. Your images are never uploaded to a server.

## How to Use

1.  **Load an Image:** Drag and drop an image file from your computer into the browser window.
2.  **Rotate:** Move the slider left or right to rotate the image to the desired angle. The image updates in real-time.
