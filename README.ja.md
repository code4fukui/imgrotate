# imgrotate

画像を回転させるための、ミニマルなクライアントサイドWebアプリです。

## デモ

**~~https://code4fukui.github.io/img~~ *(unavailable)*rotate/**

インターフェースは、大きな画像表示エリアと操作用のスライダー1つのみで構成されています。

## 機能

- **ドラッグ＆ドロップ:** ページに画像ファイルをドラッグ＆ドロップするだけで、即座に読み込めます。
- **スライダー操作:** シンプルなスライダーを使って、画像の回転を細かく調整できます。
- **広い回転範囲:** 画像を-100度から+100度まで回転させることができます。
- **クライアントサイド処理:** すべての処理はブラウザ内で完結します。画像がサーバーにアップロードされることは一切ありません。

## 使い方

1. **画像の読み込み:** パソコン内の画像ファイルをブラウザのウィンドウにドラッグ＆ドロップします。
2. **回転:** スライダーを左右に動かして、画像を任意の角度に回転させます。画像はリアルタイムで更新されます。
